## Stack:

**Front-End:**
* ReactJS
* Redux
* Stylus 
* Webpack

**Back-End**
* Node 
* Express

### Run the project locally

### Prepare the environment:

You must install:
* Node 6 
* Webpack
* BrowserSync is a global module
* Babel

### Install the dependencies:

```JS
npm install
// or
sudo npm install
```

### Start project

```JS
npm run dev
// or

npm run start

```
### lint 

```JS
npm run lint 
```

You can view the app on the url `http://localhost:4000`.
